=======
Resume
=======

Build
======

To compile run::

	pdflatex resume.tex


Spell Check
=============

::

	 aspell --lang=en --mode=tex check resume.tex

Requirements
=============

- texlive-latex-base
- texlive-latex-extra
- texlive-latex-recommended
- texlive-fonts-recommended
